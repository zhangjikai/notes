## 自动输入指令
```
#!/usr/bin/expect
set password "123"
set hostname "123"

spawn sudo poweroff
expect "\\\[sudo\\\] 123 的密码："
send "$password\r"
expect "Please type in hostname of the machine to poweroff:"
send "$hostname\r"
interact
```
